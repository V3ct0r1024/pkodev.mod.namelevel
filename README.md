## TOP/PKO/KOP mod: pkodev.mod.namelevel
Allows to display the level of the player's character next to its name. Modification for **[PKOdev.NET mod loader](https://pkodev.net/topic/5757-mod-loading-system-for-server-and-client-pkodevnet-mod-loader/)**.

## Modification information

- **Name:** pkodev.mod.namelevel;
- **Version:** 1.0;
- **Author:** V3ct0r;
- **Type:** for client (Game.exe);
- **Supported executable .exe files:** GAME_13X_0, GAME_13X_1, GAME_13X_2, GAME_13X_3, GAME_13X_4, GAME_13X_5.
